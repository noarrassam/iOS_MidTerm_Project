# iOS MidTerm Project


## Description
iOS project that implements a variety of functionalties such as ToDo List, Calendar, Facebook Login, and Fianance tabs through a segments controller.  

## Functionalities

* Prepare Function.

* Constraints.

* Cocoapods.

* TableView.

* CollectionView.

* Navigation Controller.

* TabBar Controller.
